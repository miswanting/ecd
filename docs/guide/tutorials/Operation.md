# 基本运算

## 数值运算

```erb
@EventFirst
  A = 1+2  ; 3
  B = 3-2  ; 1
  C = 2*3  ; 6
  D = 6/3  ; 2
  E = 5/3  ; 1
  F = 5%3  ; 2
  PrintVL A
  PrintVL B
  PrintVL C
  PrintVL D
  PrintVL E
  PrintVL F

  A += 2  ; 5
  PrintVL A
  A -= 2  ; 3
  PrintVL A
  A *= 2  ; 6
  PrintVL A
  A /= 2  ; 3
  PrintVL A
  A %= 2  ; 1
  PrintVL A
  Quit
```
