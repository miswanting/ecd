# 式中函数

翻译自原文档：https://osdn.net/projects/emuera/wiki/exmeth

译者：风飏@df32open in new window

Emuera 1.712追加了新的语法，被称为式中函数，或者叫“方法”（Method）。

ERB脚本中，函数是指以 @ ~ ~ 的形式定义，通过CALL命令来调用的东西。 而式中函数则是同样以 @ ~ ~ 形式定义，但直接在表达式中使用的东西。 简单的说，式中函数是可以在表达式中使用的函数。

下面演示了式中函数的使用：

A = ABS(A)
IF STRLENS(STR:0) > A
	LOCALS:0 = %SUBSTRING(STR:0, A, 1)%
ENDIF
1
2
3
4

在上面的脚本中， 令A为A的绝对值， 若STR:0的字符串长度比A大， 则令LOCALS:0为STR:0的A位置以后的截取内容。

若不使用式中函数，上面的脚本可以重写为下面：

ABS A
A = RESULT
STRLENS STR:0
IF RESULT > A
	SUBSTRING STR:0, A, 1
	LOCALS:0 = %RESULTS:0%
ENDIF

7

除了会产生中间值RESULT与RESULTS以外，这两段脚本效果完全相同。

# 示例

以下内容解释了本文用到的一些符号。 例如，

	int STRLENS(str s)
	str SUBSTRING(str s, int start = 0, int length = -1)
1
2

行首的int和str是式中函数的返回值类型。 int是数值型，str是字符串型。 下面脚本第一行是正确的，第二行是错误的。

	A = STRLENS("abc")
	A = SUBSTRING("abc", 0, 1)
	; 错误，试图为数值型变量赋予字符串型的值
1
2
3

接下来的STRLENS和SUBSTRING是式中函数的名字。

( )中的文字str s等表示式中函数的参数。 当有多个参数时，使用逗号分隔。 STRLENS有一个参数，SUBSTRING有3个参数。

参数的第一个单词表示参数的值类型。 STRLENS参数是字符串型str。 SUBSTRING的第一参数是字符串型str，第二、三参数是64位数值型int。

接下来的str、start、length的单词是参数的名字。 参数的名字通常为了方便理解参数的作用而起。

参数名字之后的 =0 表示这个参数可以省略，省略时使用的默认值。 下面脚本每一行的意思都相同：

	STR = SUBSTRING(RESULTS)
	STR = SUBSTRING(RESULTS, 0)
	STR = SUBSTRING(RESULTS, , -1)
	STR = SUBSTRING(RESULTS, 0, -1)
1
2
3
4

也可以省略第一个参数。

	;int RAND(int min = 0, int max)
	A = RAND(100)
	A = RAND( , 100)
	A = RAND(0, 100)
1
2
3
4

除此之外，式中函数也可以没有参数。

	int GETTIME()
1

虽然GETTIME没有参数，但是()是必须的。（为了区分变量和式中函数）

	int FINDCHARA(var key, ? value, int start = 0)
1

这段脚本中，var表示变量类型。这里应当输入的是TALENT等变量。 ? 表示可以接受多种类型的值。

FINDCHARA中第二参数的类型是由第一参数决定的。

	int MAX(int n，int m ...)
1

省略号表示不限制参数的数量。

	M = MAX(A，B，C，E，D，F，G）
1

M将等于A~G中数值最大的变量。
