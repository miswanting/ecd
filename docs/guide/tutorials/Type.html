#

在 Erabasic 中，值的类型大致有两种：数值和字符串。

所谓字符串，其实就是文本啦。

字符串，英文是 String，缩写是 S。

这很重要。

前面学到的变量定义#Dim语句，是定义数值的。

后面加个 S，就是定义字符串的了，比如：

@EventFirst
#DimS Text = "Hello World!"
  Quit
